Fold-outs

This feature is currently under construction

Demonstrator for displaying fold-outs (Orihon, Accordion Book) using Mirador. Hints on encoding the fold-out can be found here. the following pages are relevant:

  • IIIF cookbook
  • Example Mahābhārata scroll

Example for currently supported viewing hints:

- src: "front01.jpg"
  params:
    iiif: front01/info.json
    hint: non-paged

Example for the necessary change:

- src: "front01.jpg"
  params:
    iiif: front01/info.json
    hint: continuous
